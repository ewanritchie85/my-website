My Website repository

This website was built using HTML, CSS, and JavaScript. It is hosted from home on my Ubuntu Raspberry Pi using Nginx. It is deployed using GitHub Actions, which automatically builds and deploys the website on a Git commit. A self-hosted GitHub Actions runner on the Pi handles deployment locally, and the site is served securely over HTTPS using a Cloudflare Origin Certificate. Dynamic DNS is implemented with a scheduled script that updates my Cloudflare DNS records (ewanritchie.com and www.ewanritchie.com) every 10 minutes, ensuring the domain always resolves to my current public IP even if it changes.

Project Onyx ETL repository

For my final project of Northcoders' data engineering bootcamp, my small team created an ETL pipeline that extracted data from a Postgres OLTP database, stored it in AWS S3, processed it via Lambda functions, and loaded it into an OLAP Postgres DB. We monitored with Cloudwatch, used AWS Secrets Manager for credentials, and deployed with Terraform. Code was tested using Pytest via a GitHub Actions pipeline.
